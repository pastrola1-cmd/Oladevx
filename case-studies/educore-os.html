School Operating System

EduCore OS Platform Architecture

Architected and deployed a multi-tenant operating system for Nigerian secondary schools featuring role-based portals, automated fee invoicing, and grade books.

Client / Organization West African Educational Institutions
Role Lead Product Architect
Tech Stack Flutter Mobile App, Supabase, Next.js Admin Panel, Earnings Engine
Deployment Year 2026

The Business Problem

Educational institutions in Nigeria face high admin overheads managing paper grading records, coordinating fee reconciliations, and communicating with parents. Legacy systems are built as heavy desktop databases, making mobile-access difficult for parents and teachers.

This leads to lost records, errors in transcript compilations, and high cash-handling operational leakages.

The Solution: EduCore OS

EduCore OS is a multi-tenant school operating system developed under the OLADEVVX banner. The platform provides Flutter-based mobile portals for teachers, parents, and students, and a comprehensive Next.js web application for administrative oversight.

An integrated School Earnings Engine tracks card, transfer, and bank deposit payments, automating invoice reconciliations and matching school fee deposits instantly.

Technical Architecture

Cross-Platform Educational Hub

  • Flutter Mobile Client: Lightweight mobile access for parents to view academic progress reports and pay fees.
  • Next.js Admin Console: Management suite for school proprietors and financial auditing staffs.
  • Supabase Data Engine: Managed PostgreSQL instance handling student registries, grade rosters, and transactions.
  • Role-Based Access Control (RBAC): Multi-role configuration protecting sensitive administrative features.
